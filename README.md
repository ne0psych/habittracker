# PastelHabits Tracker

A mindful, high-performance habit tracking and time management application designed with a calming pastel aesthetic. Built with modern web technologies and a focus on user privacy.

## ‚ú® Features

- **Habit Tracking**: Manage Daily, Weekly, and Monthly habits with a clear, interactive grid.
- **Time Tracking**: Replicating the "Clockify" experience, track specific tasks linked to your habits and projects.
- **Daily Journaling**: A built-in rich text editor to document your thoughts, daily wins, and challenges with image support.
- **Deep Analytics**: Visualizations for habit consistency and streaks.
- **Privacy First**: All data is stored locally. No cloud sync, no tracking.
- **Personalization**:
  - **Dark Mode**: Toggle between a soft light pastel theme and a deep, focused dark mode.
  - **Affirmations**: Custom daily affirmations with background image personalization.
  - **Reflections**: Monthly journals to track your mental progress and growth.
- **Smart Reminders**: Desktop notifications for your daily routines.
- **Data Management**: Export and Import your entire tracking history as a JSON file.
